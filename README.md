**QuickInBox – AI Text Summarizer**

**QuickInBox** is an AI-powered text summarization web application that transforms long articles, documents, and paragraphs into short, meaningful summaries. It uses the **Hugging Face T5-Small** transformer model to generate accurate and context-aware summaries quickly and efficiently.

The application is built with **HTML**, **CSS**, and **JavaScript** for a responsive and user-friendly frontend, while the backend is developed using **Python** and **FastAPI** to process requests and interact with the T5-Small model. Users can simply paste their text into the application and receive a concise summary within seconds.

QuickInBox demonstrates the integration of modern Natural Language Processing (NLP) techniques with a lightweight web framework, providing a fast, reliable, and easy-to-use solution for students, researchers, professionals, and anyone who wants to save time by extracting the key information from lengthy text.
